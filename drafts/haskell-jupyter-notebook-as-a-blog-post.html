Haskell Jupyter Notebook as a blog post

Creating a blogpost from a Jupyter Notebook?

Why not! Let's try.

fact :: Int -> Int
fact 0 = 1
fact 1 = 1
fact n = n * fact (n - 1)

fact 10

3628800
